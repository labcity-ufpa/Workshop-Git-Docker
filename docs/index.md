# Aula completa de Git e Docker

Um guia progressivo para aprender Git e Docker desde a instalação até fluxos profissionais.

## Bem-vindo!

Neste material, você aprenderá de forma prática e estruturada as duas ferramentas mais fundamentais para o desenvolvimento de software moderno: **Git** (controle de versão) e **Docker** (contêinerização).

### O que você vai aprender?

- **Git:** Como iniciar projetos, versionar código, entender o modelo mental (Working Directory, Staging, Repository), usar repositórios remotos, resolver conflitos e usar fluxos profissionais como o **Git Flow**.
- **Docker:** Por que e como usar contêineres, diferença para máquinas virtuais, instalação, criação de imagens com `Dockerfile`, subir infraestrutura inteira com `Docker Compose` e boas práticas.
